# VC_Tacotron

Voice Conversion using Tacotron Model.

### Quick Start
#### 1. Download dataset and Preprocess
```shell
./download_data.sh
python preprocess.py
```


#### 2. Training
``` shell
python train.py
```

#### 3. Evaluation
```shell
./eval.sh
```

### Results
#### 1. Audio samples
clb -> dbl
